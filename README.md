# node-red-contrib-onem2m

在 Node-RED 上使用 oneM2M 協定

use onem2m protocol at Node-RED

## How install 如何安裝

請先到需要安裝該 node 的 Node-RED 資料夾

`$ cd [您安裝的位置]/.node-red`

使用 npm 安裝該 node

`$ npm install node-red-contrib-onem2m`

重新打開 node-red，即可看到安裝的 node

`$ node-red`

## Nodes 節點

目前完成的 oneM2M 資源 node 如下

- CSE
- AE
- Container
- contentInstance
- subscription

能夠使用的 protocal

- http
